# Greencastle Brand Guidelines

*A botanical-naturalist design system for environmental education, ethnobotany, and nature-focused content*

---

## Brand Essence

**Greencastle** evokes the quiet authority of a Victorian herbarium meets modern environmental science. It balances academic credibility with warm approachability—scholarly but never stuffy, earthy but refined.

**Voice:** Knowledgeable, warm, grounded, curious
**Mood:** Like afternoon light in a botanical library

---

## Color Palette

| Role | Name | Hex | Usage |
|------|------|-----|-------|
| **Primary** | Botanical Green | `#3D5A3D` | Headers, primary buttons, accent bars, key emphasis |
| **Secondary** | Sienna | `#8B5A2B` | Subheadings, secondary buttons, warm accents, links |
| **Accent** | Terracotta | `#C75B3A` | Calls to action, warnings, highlights, energy points |
| **Surface** | Parchment | `#F8F5F0` | Backgrounds, cards, content areas |
| **Surface Alt** | Warm Sand | `#E8E0D5` | Secondary backgrounds, boxes, muted containers |
| **Muted** | Stone | `#5C574F` | Body text, secondary text |
| **Border** | Clay | `#C4B8A8` | Borders, dividers, subtle separations |
| **Dark** | Charcoal Earth | `#2D2A26` | Dark backgrounds, footer blocks, contrast areas |

**Accent Variations:**
- Soft Green: `#6B8E6B` — success states, positive indicators
- Dusty Purple: `#6B4C7A` — specialty categories, creative elements
- Warm Cream: `#F5EDE3` — highlight boxes, callouts

## Typography

### Display (Headings)
**Font:** Georgia (serif)
**Weight:** 700 (Bold)
**Character:** Traditional, authoritative, readable

| Level | Size | Color | Usage |
|-------|------|-------|-------|
| H1 | 36–52px | Botanical Green | Page titles, hero text |
| H2 | 28–36px | Botanical Green | Section headers |
| H3 | 20–24px | Sienna | Subsections, card titles |

### Body (Content)
**Font:** Arial (sans-serif)
**Weight:** 400 (Regular), 600 (Semibold for emphasis)
**Size:** 14–16px
**Line Height:** 1.4–1.5
**Color:** Stone `#5C574F` or Charcoal Earth `#2D2A26`

### Accent Text
- **Labels/Tags:** 10–12px, uppercase, letter-spacing 2–3px, Sienna
- **Footnotes:** 10px, Stone, italic for sources
- **Callouts:** 13–14px, semibold, Terracotta

---

## Spacing System

Base unit: **4px**

| Token | Value | Usage |
|-------|-------|-------|
| `xs` | 4px | Tight text spacing |
| `sm` | 8px | Related elements |
| `md` | 12px | Default gaps |
| `lg` | 16px | Section padding |
| `xl` | 20–24px | Major separations |
| `2xl` | 32px | Hero spacing |

**Consistent rhythm:** Use multiples of 4px for all spacing decisions.

---

## Component Patterns

### Cards/Boxes
- Background: Warm Sand `#E8E0D5`
- Border-radius: 6–8px
- Border-left accent: 5px solid (Primary, Secondary, or Accent)
- Padding: 14–16px

### Buttons
- Primary: Botanical Green bg, white text, 6px radius
- Secondary: Sienna bg, white text
- Accent: Terracotta bg, white text

### Tables
- Header: Primary color bg, white text, centered
- Rows: Alternating Parchment / Warm Sand
- Borders: 1px Clay

### Decorative Elements
- Vertical accent bar: 8–12px wide, gradient from Botanical Green to Sienna
- Horizontal divider: 3px solid, Terracotta or Sienna, 80–120px wide
- Corner accents: L-shaped borders in primary colors

---

## Application Examples

**Presentations:** Left accent bar (8px), Georgia titles, earth-toned cards, botanical green headers
**Documents:** Clean hierarchy, sienna subheads, parchment callout boxes, footer with Stone text
**Web:** Warm sand backgrounds, green primary CTAs, terracotta for urgent actions

---

## Do / Don't

✓ Use generous whitespace — let content breathe
✓ Pair serif headers with sans-serif body
✓ Use accent colors sparingly for emphasis
✓ Maintain warm, approachable tone

✗ Don't use pure black (`#000000`) — use Charcoal Earth instead
✗ Don't overuse Terracotta — reserve for key moments
✗ Don't combine multiple accent colors in one element
✗ Don't use cold blues or grays — stay in warm earth tones
